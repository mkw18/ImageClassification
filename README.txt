采用pytorch1.7.1开发，主要硬件加速器版本是cuda10.1
主文件为ImgClassification.ipynb；
model文件夹中是自己实现的神经网络；
plot是训练结果绘制的图像，在报告中有包含；
test_record是测试过程中的记录，主要是自测集上各类别的正确率和投票结果；
train_record是训练过程中的记录，记录的是尝试不同方法的训练和验证结果，可能比较乱，并且由于训练网络的过程是在多个平台上进行的，所以不太全，报告中有总结；
util是程序依赖的一些函数；
validate_record是训练过程中在验证集上具体到每一类的正确率，便于分析每种神经网络对于不同类别的分类效果好坏；
weights是训练得到的部分权重，此处只包含了最后用得上的权重；
reference.txt中记录了我在完成任务的过程中主要的参考文献。
